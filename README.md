# react-native-mlkit-barcode
Google mlkit based barcode-qr code scanner
## Installation

```sh
npm install react-native-mlkit-barcode
```

## Usage

```js
import { MlkitBarcodeView } from "react-native-mlkit-barcode";

// ...

<MlkitBarcodeView color="tomato" />
```
